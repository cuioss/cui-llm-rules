# Script Runner Skill

Generates the scripts library cache (`.plan/scripts-library.toon`) from marketplace inventory.

**Note**: This skill is for cache generation only. Script path resolution is a **system convention** documented in CLAUDE.md, not a skill workflow.

## Background

Claude Code skills cannot reliably execute bundled scripts because:
- Relative paths like `scripts/foo.sh` resolve from current working directory, not skill directory
- Absolute paths are machine-specific and not portable

## Solution

This skill generates a cache file that maps portable notation to absolute paths:
1. **Portable notation**: `bundle:skill/scripts/name.ext` format
2. **Cache generation**: Creates `.plan/scripts-library.toon` with all script mappings
3. **TOON format**: Token-efficient tabular format

## Notation Format

```
bundle:skill/scripts/name.ext

Examples:
- cui-plugin-development-tools:marketplace-inventory/scripts/scan-marketplace-inventory.py
- general-tools:permission-management/scripts/generate-permission-wildcards.py
```

See `references/notation-spec.md` for full specification.

## Storage

Script mappings are stored in `.plan/scripts-library.toon`:
- Machine-specific (absolute paths)
- Covered by `.plan/` in `.gitignore`
- Generated by `/tools-discover-skill-scripts` command

## Workflows

### Workflow: Discover

Discovers all skill scripts from installed plugins and generates the scripts cache in TOON format.

**Script**: `general-tools:script-runner/scripts/generate-scripts-library.py`

#### Steps

1. **Run discovery script**:
   ```bash
   python3 {absolute_path_to_generate_scripts_library} --marketplace-root {workspace-root}
   ```

   This script:
   - Invokes `scan-marketplace-inventory.py` to get all bundles, skills, and scripts
   - Builds portable notation for each script: `{bundle}:{skill}/scripts/{script-name}`
   - Writes `.plan/scripts-library.toon` atomically

#### Output

TOON response from script:
```toon
status: success
scripts_discovered: 81
output_file: .plan/scripts-library.toon
marketplace: cui-development-standards
```

#### Generated File Format

```toon
version: 2
generated: 2025-12-07T10:30:00Z
marketplace: cui-development-standards
script_count: 81

scripts[81]{notation,absolute,type}:
builder:builder-maven-rules/scripts/execute-maven-build.py,/full/path/execute-maven-build.py,python
planning:manage-files/scripts/manage-files.py,/full/path/manage-files.py,python
...
```

## Script Resolution (System Convention)

Script path resolution is a **system convention**, not a skill workflow. See `CLAUDE.md` for the authoritative documentation.

**Quick reference**:
1. Skills document scripts using portable notation
2. When executing, resolve the absolute path from `.plan/scripts-library.toon`
3. No skill loading required - resolution is built into the system
