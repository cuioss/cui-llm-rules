# Workflow base directory hardcoded in 45 files instead of single config

## Detail

Renaming .cui/ to .plan/ required changing 45 files across multiple bundles. While Python scripts use base_path() abstraction, documentation hardcodes the actual path. Future: (1) Define directory name as a constant in one place, (2) Reference via variable in docs where possible, (3) Consider config file for path. The Python abstraction (base_path, get_base_dir, set_base_dir) was correct but documentation spread the literal path everywhere.
